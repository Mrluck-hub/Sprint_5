# Тестирование сайта https://stellarburgers.education-services.ru/

В проекте реализованы тесты с использованием фреймворка `pytest`. Тесты расположены в папке `tests`

## Проект

### Тесты

- Проверяет успешную регистрацию с валидными данными (`test_registration.py`)
- Проверяет вход в аккаунт с разных кнопок (`tes_log_in.py`)
- Проверяет выход из аккаунта (авторизованного) (`test_personal_cabinet_exit.py`)
- Проверяет переход к конструктору (`test_click_constructor_logo.py`)
- Проверяет переключение разделов в конструкторе (`test_construction_tabs.py`)
